## MBLOG

> mblog (mtons blog)开源博客系统,是一个简单、大气、美观、支持多端（PC,Mobile,Pad）访问的的多人博客系统, 采用Java语言开发,基于MySQL数据库;具有性能优异、简单实用、安全稳定、支持全文检索的特点。

### 技术选型：

* JDK8
* SSH (Spring、SpringMVC、Hibernate）
* 安全权限 Shiro
* 搜索工具 Lucene
* 缓存 Ehcache
* 视图模板 Velocity
* 其它 Jsoup、fastjson、GraphicsMagick
* jQuery、Seajs
* Bootstrap 前端框架
* UEditor/Markdown编辑器
* font-wesome 字体/图标
